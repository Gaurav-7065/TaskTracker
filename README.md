✨ Features

Core (Mandatory)

✅ Create Tasks — Add tasks with title, description, status, priority, and due date
✅ View Tasks — All tasks displayed in a responsive card grid
✅ Update Tasks — Edit any task inline via modal form
✅ Delete Tasks — Remove tasks with confirmation
✅ Form Validation — Frontend + backend validation (title required, schema enforced)
✅ REST API — Proper HTTP methods (GET, POST, PUT, DELETE) with correct status codes
✅ MongoDB Integration — Mongoose schema with Atlas cloud database
✅ Responsive UI — Works on mobile, tablet, and desktop
✅ Dynamic Updates — No page refresh needed — React state handles all updates

✨ Extra Feature:
1.Toogle theme
2.react-hot-toast for notification
3.Status badges + priority indicators
